1 Description
--------------------------------------------------------------------------------

GSXSteg is a small commandline program which provides steganography compatible
with GPGSX <http://web.utanet.at/ascherst/gpgsx/>. This program is for people who
do not have GPGSX installed but got a steg object created with GPGSX, so they do
not need to install GPGSX. But it may also be used to hide messages with the same
algorithm as GPGSX.

This program is part of GPGSX but may be distributed independently.

The source code of this program is included in the source distribution of GPGSX,
you can download it from http://web.utanet.at/ascherst/gpgsx/.

3 Requirements/Installation
--------------------------------------------------------------------------------

This program does not have any requirements and installation is not needed. Just
copy it to some folder and run it from there.

--------------------------------------------------------------------------------
4 Commandline arguments
--------------------------------------------------------------------------------

Usage:
  gsxsteg [-batch] -<action> [-<argument> <value>]
 
  batch: Batch mode, everything must be given with commandline arguments
action can be:
  hide: Hide data inside a medium
  extract: Extract data from a medium
argument can be:
  medium: The medium file
  dest: The destination file
  msg: Filename with message to hide
  pass: Password
Example:
  gsxsteg -batch -extract -medium mymedium.png -dest thedata.txt -pass "my secret pass"
